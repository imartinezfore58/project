## 🔧 Easy Customization

### Update Content
1. Open any `.jsx` file in `src/components/`
2. Find the text you want to change
3. Update it
4. Save (portfolio auto-reloads)

### Change Colors
1. Open `src/index.css`
2. Update the `--primary-color` and `--secondary-color`
3. Save

### Add Social Links
1. Open `src/components/Contact.jsx`
2. Update social media URLs
3. Save
